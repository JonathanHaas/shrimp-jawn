# 🦐 Shrimp Jawn

A whimsical, single-page shrimp-centric website built for and inspired by [@shrimpjawn](https://www.instagram.com/shrimpjawn) out of Fishtown, Philadelphia.

## What's in the site

- **Hero** — animated bobbing shrimp, gradient ocean background, floating bubble particles
- **Scrolling marquee** — shrimp facts ticker
- **Facts grid** — mind-blowing shrimp science cards
- **Shrimp of the World** — species showcase
- **Poetry Corner** — odes and haikus dedicated to shrimp
- **Greatest Hits** — recipe cards
- **Shrimp Ate Counter** — live billions counter with speed modes
- **FAQ** — accordion with burning shrimp questions
- **About drawer** — hamburger menu with photo and shrimp enthusiast bio
- **Character bubbles** — Harry Styles, Bubba, and Pepe the King Prawn floating in the corner

## Stack

Plain HTML, CSS, JavaScript. No frameworks, no build step. One file: `index.html`.

## Deploy

### Cloudflare Pages (recommended)
1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Create project → Upload assets
3. Drop `index.html` in
4. Done — get a free `*.pages.dev` URL

### GitHub Pages
1. Push `index.html` to a GitHub repo
2. Settings → Pages → Deploy from branch → main → / (root)
